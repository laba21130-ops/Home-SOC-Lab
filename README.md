# Home SOC Lab

## Project Overview

This project documents the creation of a Home SOC (Security Operations Center) lab environment using VirtualBox, Kali Linux, Windows, Splunk, and Sysmon.

The purpose of this lab is to develop hands-on cybersecurity and blue team skills through attack simulation, log analysis, and threat detection.

---

# Phase 1 — Kali Linux Virtual Machine Setup

## Objective

The objective of this phase was to create an attacker machine using Kali Linux within VirtualBox. This system will later be used to simulate attacks and generate security events for monitoring and detection within the SOC environment.

---

## Tools Used

* Oracle VirtualBox
* Kali Linux

---

## Steps Performed

1. Downloaded Kali Linux VirtualBox image
2. Imported the VM into VirtualBox
3. Booted the Kali Linux virtual machine
4. Logged into the system using default credentials
5. Verified successful installation and operation

---

## Lessons Learned

* Learned how to import and configure a virtual machine in VirtualBox
* Became familiar with Kali Linux initial setup
* Understood the role of attacker systems within a SOC lab environment
