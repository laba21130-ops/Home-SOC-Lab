# Home SOC Lab

## Project Overview

This project documents the creation of a Home SOC (Security Operations Center) lab environment using VirtualBox, Kali Linux, Windows, Splunk, and Sysmon.

The purpose of this lab is to develop hands-on cybersecurity and blue team skills through attack simulation, log analysis, and threat detection.

---

# Phase 1 — Windows  Virtual Machine Setup

## Objective

The objective of this phase was to create an victim machine using Windows within VirtualBox. This system will later be used to simulate attacks and generate security events for monitoring and detection within the SOC environment.

---

## Tools Used

* Oracle VirtualBox
* Windows 11 ISO

---

## Steps Performed

1. Downloaded Windows VirtualBox image
2. Imported the VM into VirtualBox
3. Booted the Windows virtual machine
4. Logged into the system using default credentials
5. Verified successful installation and operation

---

## Lessons Learned

* Learned how to import and configure a virtual machine in VirtualBox
* Became familiar with Windows initial setup
* Understood the role of Victim machine ithin a SOC lab environment
